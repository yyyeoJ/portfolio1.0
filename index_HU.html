React
Tailwind
Reszponzív

Egyéb infók

A MealSpiration egy receptkereső webes alkalmazás
Egy publikus api segítségével kéri le az adatokat, így a hívások száma limitált
Betöltéskor véletlenszerű recepteket dob fel, majd később kereshetünk másra
A találatok szűrhetők étel típus, diéták, és allergének alapján
Minden recepthez makro tápanyagok

Amit tanultam:
• React alapok
• API használat
• JSON adathalmaz manipulálása
• A kívánt adatok reprezentálása funkcionális React komponensként
